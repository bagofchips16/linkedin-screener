What is LinkedIn Screener?

LinkedIn Screener is a free tool that helps HR professionals and recruiters quickly evaluate whether a candidate is a good fit for a job role. Simply define what you're looking for, paste a candidate's LinkedIn profile text, and get an instant score with a detailed breakdown.

🎯

Define Your Role

Set up required skills, experience range, location, and keywords for any position.

📋

Paste & Score

Copy-paste a LinkedIn profile and get an instant relevance score (0-100%).

📊

Bulk Screen

Upload a CSV with multiple candidates and rank them all at once.

🔒

Private & Secure

Runs on your own computer. No data is sent to any server or third party.

How to Install (One-Time Setup)

You only need to do this once. It takes about 2 minutes.

Install Python

Download and install Python from python.org/downloads. During installation, make sure to check the box that says "Add Python to PATH".

💡 If you already have Python installed, you can skip this step.

Download the Tool

Download the LinkedIn Screener from GitHub:

# Option 1: Using git
git clone https://github.com/bagofchips16/linkedin-screener.git

# Option 2: Download ZIP from GitHub and extract it
# Go to: github.com/bagofchips16/linkedin-screener
# Click green "Code" button → "Download ZIP"

Install Dependencies

Open a terminal (Command Prompt or PowerShell) in the downloaded folder and run:

cd linkedin-screener/backend
pip install -r requirements.txt

Start the Tool

Run this command to start the screener:

python -m uvicorn main:app --port 8000

Then open your browser and go to: http://localhost:8000

💡 Every time you want to use the tool, just run this command from the backend folder.

How to Use (Step by Step)

Create a Role

Click the "Define Roles" tab. Fill in:

  • Role Title — e.g. "Senior Frontend Engineer"
  • Required Skills — Type each skill and press Enter (e.g. React, JavaScript, CSS)
  • Preferred Skills — Nice-to-have skills
  • Experience — Min/max years required
  • Location — Where you want the candidate (optional)
  • Keywords — Extra words to look for (e.g. "startup", "SaaS")

Click "Create Role" when done.

Screen a Single Candidate

Click the "Screen Candidate" tab. Then:

  1. Select the role you created from the dropdown
  2. Go to the candidate's LinkedIn profile
  3. Select all the text on their profile (Ctrl+A) and copy it (Ctrl+C)
  4. Paste it (Ctrl+V) into the text box
  5. Click "Screen Candidate"

You'll instantly see a score and breakdown!

Screen Many Candidates at Once

Click the "Bulk Upload" tab. Prepare a CSV file (spreadsheet) with these columns:

name,headline,location,skills,experience_years,education,profile_text
John Smith,Senior Engineer,Bangalore,"python,react,aws",5,B.Tech IIT,Full profile text
Jane Doe,Product Manager,Mumbai,"product management,agile",7,MBA IIM,Strategic PM...

Select your role, upload the file, and all candidates will be scored and ranked.

💡 You can create this CSV in Excel or Google Sheets — just save/export as CSV.

Review Results

Click the "Results" tab to see all screened candidates ranked from best to worst match. Click on any candidate to expand their detailed score breakdown.

Understanding the Scores

Each candidate gets a score from 0-100% based on how well they match your role requirements.

Score Categories

CategoryWeightWhat it Measures
Required Skills40%How many must-have skills the candidate has
Experience25%Whether their years of experience match the range
Keywords15%Whether your custom keywords appear in their profile
Location10%Whether they're in the desired location
Preferred Skills10%How many nice-to-have skills they have

What the Ratings Mean

80-100% — Strong Match Excellent fit. Prioritize for interview.

60-79% — Good Match Solid candidate. Worth considering.

40-59% — Partial Match Some relevant skills but gaps exist.

0-39% — Weak Match Not a good fit for this role.

Tips for Best Results

Be Specific with Skills

Instead of "programming", list specific skills like "Python", "React", "SQL". The tool matches exact words.

Copy Full LinkedIn Profiles

The more text you paste, the better the scoring. Include their About section, Experience, Skills, and Education.

Use Keywords Wisely

Add industry terms, company types, or domain keywords like "fintech", "healthcare", "B2B SaaS" to find domain-relevant candidates.

Create Multiple Roles

You can screen the same candidate against different roles to find the best fit across your open positions.

Frequently Asked Questions

Is my data safe?

Yes! The tool runs entirely on your computer. No candidate data is ever sent to any external server or stored permanently. When you close the tool, all data is cleared.

Do I need internet to use this?

Only for the initial setup (downloading and installing). After that, the tool runs completely offline on your machine.

Can I use this on Mac or Linux?

Yes! The tool works on Windows, Mac, and Linux. Just make sure Python is installed.

Does this access LinkedIn directly?

No. You manually copy-paste profile text. The tool does not connect to LinkedIn or scrape any data.

How accurate is the scoring?

The tool uses keyword matching, which works well for filtering obvious matches and mismatches. It's meant to save time on initial screening — always do a human review before making hiring decisions.

Is it free?

100% free and open source. No subscriptions, no limits, no hidden costs.
